# 文件分类整理工具

一个简单的Python脚本，能把文件夹里的文件按类型分类。

## 能分哪些类
- 图片: jpg, png, gif 等
- 文档: pdf, docx, xlsx, txt 等
- 音视频: mp4, mp3, avi 等
- 压缩包: zip, rar 等
- 代码: py, js, html, css 等
- 其他: 不在上面的都放这里

## 怎么用
把代码下载到电脑上，打开命令行输入:
python file_organizer.py (要整理的文件夹路径)
也可以直接双击运行，再输入路径。
